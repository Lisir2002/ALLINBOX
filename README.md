# all-in-box

ALL IN BOX - 综合工具箱应用

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

## 项目结构

```
all-in-box/
├── lib/
│   ├── models/        # 数据模型
│   ├── screens/       # 页面
│   ├── widgets/       # 组件
│   ├── services/      # 服务
│   └── utils/         # 工具函数
├── .gitignore
└── README.md
```

## 技术栈

- Flutter + Dart
